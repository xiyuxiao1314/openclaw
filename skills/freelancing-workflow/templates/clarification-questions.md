# Clarification Questions Template

## General

1. What is the exact deliverable you want?
2. Do you have reference files, examples, or supporting documents?
3. What counts as complete for this task?
4. What is included, and what is explicitly not included?
5. What is your expected timeline?
6. Should this stay limited to the currently described scope?

## Scope Control

1. Is this limited to the specific issue or deliverable already described?
2. Should extra features or adjacent work be excluded for now?
3. If new complexity appears, should it be re-confirmed separately?

## Coding Task Add-On

1. What language, framework, or runtime is involved?
2. What is the current behavior?
3. What is the expected behavior?
4. Can you provide the relevant file, snippet, or reproducible sample?
5. Does this require deployment, server access, or privileged access?

## Translation / Text Add-On

1. What is the source language and target language?
2. What is the intended use of the text?
3. Who is the target audience?
4. What tone or style do you want?
5. Do you have a terminology list or brand voice requirement?
6. Is this translation only, or translation plus polishing/localization?
