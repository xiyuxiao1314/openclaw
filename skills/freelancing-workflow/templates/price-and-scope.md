# Price and Scope Template

## Scope Confirmation

Before confirming price, make sure the following are clear:

- exact deliverable
- what is included
- what is not included
- whether this remains a small task
- whether any hidden complexity is likely

## Timing Guardrails

Use bounded language:

- “Based on the current scope...”
- “If no additional complexity appears...”
- “For this defined task...”

Avoid absolute promises when uncertainty remains.
