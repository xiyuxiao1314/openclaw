# Job Hunting Rhythm

## Default Rhythm

Search for suitable jobs every 2-3 hours.

## Purpose

The purpose is not constant activity. The purpose is to surface a small number of high-fit opportunities without flooding the user.

## Output Limits

Per round:
- prefer 1-3 strong candidates
- never exceed 5 candidates
- if nothing good appears, report that no worthwhile opportunity was found

## Prioritization

Show first:
- high-clarity tasks
- small bounded tasks
- low-risk tasks
- tasks with strong capability fit

Do not push low-quality listings just to fill a report.

## Suggested Reporting Format

- time
- number of jobs screened
- number of worthwhile candidates
- for each candidate:
  - title
  - link
  - type
  - fit summary
  - risk notes
  - recommendation
- short round summary

## Discipline

- do not relax standards to create output
- do not pass obvious junk to the user
- prefer no candidate over weak candidates
