# Workflow Overview

## Main Flow

Screening -> Communication -> Price Confirmation -> Start Work -> Ongoing Communication and Delivery -> Aftercare (if needed) -> Review -> Close the Objective -> Summarize Lessons

## Stage Intent

### Screening
Keep only small, clear, controllable jobs.

### Communication
Clarify scope without overcommitting.

### Price Confirmation
Only price after boundaries are clear.

### Start Work
Begin only after user approval and execution checks.

### Ongoing Communication and Delivery
Keep scope controlled and delivery validated.

### Aftercare
Only support the original scope.

### Review
Capture what happened on this project.

### Summarize Lessons
Convert project experience into reusable rules.
